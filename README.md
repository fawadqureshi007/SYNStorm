## 💀 About SYNStorm
SYNStorm is a **TCP SYN flood simulation tool** designed for **ethical hackers, cybersecurity enthusiasts, and penetration testers** to understand the mechanics of network stress testing in a controlled lab environment.

It provides a **CLI interface** inspired by Kali tools like Metasploit, with animated banners and hacker-style prompts.

---

## ⚡ Features
- Hacker-style **ASCII banner**  
- CLI menu for launching tools  
- DDOS tool launcher (educational only)  
- Works on **Linux (Kali)** and **Windows**  
- Modular design – easy to add new attack modules  
- Written entirely in **Python 3**  

---

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/SYNStorm.git
cd SYNStorm
````

Install required libraries:

```bash
pip3 install -r requirements.txt
```

Make scripts executable (Linux only):

```bash
chmod +x synstorm.py
chmod +x files/ddos/ddos.py
```

---

## 🚀 Usage

Run the tool:

```bash
python3 synstorm.py
```

Example session:

```
SYNStorm@h4cker_fawad:~# 1
Launching DDOS Tool...
```

Press `2` in the menu for **tool info**, `3` to **exit**.
